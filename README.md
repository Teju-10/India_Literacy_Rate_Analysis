# India Literacy Rate Analysis

## Project Overview

This project focuses on analyzing the literacy rates across different states and union territories of India using **Power BI**. The goal is to understand the distribution of **male, female, and average literacy rates** and present the insights through an interactive dashboard.

---

## Dataset Information

The dataset used in this project contains literacy data for **37 states and union territories** of India.

### Columns Included:

* States & Union Territories
* Male Literacy Rate (%)
* Female Literacy Rate (%)
* Average Literacy Rate (%)

---

## Data Preparation

Before building the dashboard, the data was cleaned and transformed:

1. Converted raw text data into structured format using **Text to Columns** in Excel
2. Ensured proper column separation and formatting
3. Checked for consistency and accuracy in values
4. Saved the cleaned dataset in Excel format

---

## Power BI Implementation

The cleaned dataset was imported into Power BI to create an interactive dashboard.

### Visuals Created:

* **KPI Cards** → Total Male Literacy & Female Literacy
* **Bar Charts** → State-wise Male and Female Literacy Comparison
* **Map Visualization** → Literacy distribution across India
* **Pie Chart** → Overall Male vs Female Literacy Ratio
* **Filters/Slicers** → To view data for specific states

---

## Key Insights

* Male literacy rate is generally higher than female literacy across most states
* Some states show a smaller gap between male and female literacy
* Regional differences in literacy can be clearly observed through the map
* Overall literacy trends help identify areas needing improvement

---

## Tools & Technologies Used

* Microsoft Excel (Data Cleaning & Preparation)
* Power BI (Dashboard Creation & Visualization)

---

## Conclusion

This project helps in understanding literacy patterns in India and highlights the gap between male and female literacy rates. The dashboard makes it easy to explore and compare data across different states.
